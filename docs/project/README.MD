# 💼 Arquivos do Projeto

Este diretório contém arquivos referentes ao gerenciamento e planejamento do projeto, bem como um diário de bordo para registro semanal do andamento do projeto.

## 📁 Estrutura do Diretório
```
project/
├── bs-beauty-academy-project.pod          # Planejamento do projeto
├── README.MD                              # Este arquivo
└── work-log.md                            # Diário de bordo do projeto
```

## ⚙️ Usando o Project Libre
Para abrir e utilizar o arquivo ``bs-beauty-academy-project.pod`` usando o Project Libre, siga os seguintes passos:
- Baixe o software através [deste link](https://sourceforge.net/projects/projectlibre/).

- Realize o download do arquivo no [formato ``.pod``](./bs-beauty-academy-project.pod).

- Com o software instalado, abra o Project Libre e siga as instruções abaixo:
     - Clique em "Abrir Projeto".
     - Procure e selecione o arquivo `.pod` baixado previamente.
